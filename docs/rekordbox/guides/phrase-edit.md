# 1 Before Start

## 1.1 Before getting started

This Operation Guide explains how to edit phrases, created from Phrase Analysis in rekordbox.
For instructions on rekordbox in general, please see the rekordbox Operating Instructions.

rekordbox Instruction Manual    rekordbox.com/en/download/#manual

## 1.2 Phrase Analysis

In rekordbox, you can analyze the structure of a track through Phrase Analysis. After Phrase Analysis, each part of the structure is defined as a phrase. and the track structure is expressed as a set of the following phrases.

| Phrase | Description |
|---|---|
| INTRO | an opening phrase of a track |
| UP | a build-up phrase of a track |
| DOWN | a breakdown phrase of a track |
| CHORUS | an uplifting phrase of a track |
| BRIDGE | an interlude phrase of a track |
| VERSE | a phrase that does not apply to other phrases |
| OUTRO | an ending phrase of a track |

For Phrase Analysis, please see the rekordbox Operating Instructions.

rekordbox Instruction Manual    rekordbox.com/en/download/#manual

## 1.3 Display results of Phrase Analsys

Load a track analyzed by Phrase Analysis to a deck (Deck1~4). Result from Phrase Analysis is shown under the waveform on the track information panel.

\*Open [Preferences] > [View] category > [Layout] tab > [Phrase (Full Waveform)] and tick/untick to show/hide phrase analysis results.

[Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood at 125.00 BPM in Em key, with phrase analysis bar below the full waveform displaying colored phrase labels including INTRO 1, UP 1, UP 1, CHOR..., DO, U..., UP 1, CHOR..., DO, U..., UP 1, U, CHOR..., CHOR..., CH..., C, DO, O. Below are hot cue pads A-H and transport controls.]

When you select [Phrase (Enlarged Waveform)], phrase analysis result is shown in the same way as below.
\*Open [Preferences] > [View] category > [Layout] tab > [Phrase (Enlarged Waveform)] and tick/untick to show/hide phrase analysis results.

[Screenshot: Enlarged waveform view showing a phrase labeled "CHORUS 1" with "179.1Bars" indicator, with colored waveform detail above the phrase bar.]

\* A striped pattern is shown if [Fill in] is included in the phrases. (Enlarged Waveform only).

\* [Fill in] is a section that provides improvisational changes at the end of phrase. [Fill in] is detected at the end of Intro, Up, and Chorus (up to 4 beats).

# 2 Phrase Edit

To edit phrases, click the edit icon on the deck to display [PHRASE EDIT].
Only one deck can be edited at one time (Deck 1~4).

[Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood at 125.00 BPM, with GRID EDIT on the left and PHRASE EDIT panel on the right containing CUT button, phrase type selector, and CLEAR button. Below are grid editing controls including beat grid markers, TAP, x2, x1/2 buttons, and waveform zoom controls.]

\*NOTE: If you perform Phrase Analysis again for the track of which you edited phrases, your edit will be cleared.

## 2.1 Adjust positions and combine phrases

You can adjust the position of a phrase or combine phrases.

[Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood at -00:41.4 / 05:57.9 with phrase analysis bar. Two yellow bracket markers are visible at both ends of a selected phrase on the waveform. PHRASE EDIT panel shows CUT button, CHORUS 1 selector, and CLEAR button.]

1. Click a phrase. (e.g. CHORUS)
   A bracket marker appears at both ends of the phrase.

2. Click the bracket marker to move backward or forward and release the mouse.
   The split position of the phrase will be changed.
   If the bracket marker is moved beyond another phrase, the phrases will be combined.

\*You can do the same on enlarged waveforms.

[Screenshot: Enlarged waveform view showing a phrase labeled "CHORUS 1" with a bracket marker visible, and "180.2B" bars indicator.]

\*NOTE: If [Fill in] is moved from the end of the phrase or divided due to the edit, the [Fill in] will be deleted.

## 2.2 Cut a phrase

You can cut a phrase and divide it into two phrases. If a phrase is divided, two phrases of the same type will be created. You can change the type of the phrase later. (To change the type, see 2.3 *Change phrase type*.)

[Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood with PHRASE EDIT panel. The CUT button is highlighted in blue, indicating cut mode is active. CLEAR button shown at bottom.]

1. Click [CUT] (black).
   The CUT icon turns blue. [CUT] (The CUT feature is enabled.)

2. Move the mouse to a phrase you would like to cut. (e.g. CHORUS)
   A scissors icon appears on the phrase.

3. Click the scissors icon where you would like to cut.
   The phrase will be divided.

\* You can do the same on enlarged waveforms.

\* NOTE: If [Fill in] is divided due to the edit, the [Fill in] will be deleted.

## 2.3 Change phrase type

You can change types of phrases detected in phrase analysis.

[Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood with PHRASE EDIT panel. CUT button, phrase type dropdown showing "CHORUS 1", and CLEAR button are visible.]

1. Click a phrase on the bar (CHORUS) and then click [CHORUS 1] dropdown button under [CUT] button to show the drop-down menu below.

   | Drop-down menu |
   |---|
   | INTRO 1 |
   | INTRO 2 |
   | UP 1 |
   | UP 2 |
   | UP 3 |
   | CHORUS 1 (selected) |
   | CHORUS 2 |
   | DOWN |
   | OUTRO 1 |
   | OUTRO 2 |

2. Select a type from the drop-down menu to change the type.

\* You can do the same on enlarged waveforms.

\*NOTE: If [Intro]/[Up]/[Chorus] is changed to a different type and [Fill in] in included before the edit, the [Fill in] will be deleted.

## 2.4 Clear phrases

If you clear phrases, results of Phrase Analysis will be deleted except [Intro]. Then you can edit by cutting a phrase or changing types to create phrases as you like.

[Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood with PHRASE EDIT panel. CUT button, CHORUS 1 dropdown, and CLEAR button (highlighted in blue) are visible.]

1. Click [CLEAR] to show the dialog below.

   [Screenshot: Dialog box with message "Are you sure you really want to clear all the analyzed phrases? When cleared, all the analyzed phrases are deleted." with Yes and No buttons.]

2. Click [Yes] to clear all the analyzed phrases.
   All the phrases are cleared except [Intro].

   [Screenshot: Deck view showing track "Serenade (Ki Creighton & Makanan Remix...)" by Sirus Hood after clearing phrases. Only a single red INTRO 1 phrase spans the entire track length. PHRASE EDIT panel shows CUT button, empty phrase type selector, and CLEAR button.]

\* NOTE: If you clear the phrases, the [Fill in] will be deleted.
