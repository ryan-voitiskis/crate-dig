# Introduction

On EDIT mode, you can edit tracks in advance to make your DJ performance easier. You can easily change the track composition such as extending intro/outro, copying/deleting the interlude.
Depending on your subscription plan, some functions may not be available. For details on subscription plan, refer to the rekordbox website.
rekordbox.com/en/

[Screenshot: subscription plan icon] : Invalid function on your subscription plan

If you have used an earlier rekordbox version than ver. 6.1.0, EDIT mode is not available. Please update rekordbox to the latest version (ver. 6.1.1 or above).

On Edit mode, following contents cannot be edited.

- Video files
- Streaming tracks

# About the EDIT mode screen

Check that [**EDIT**] is selected in the upper-left corner of the screen.

[Screenshot: Full EDIT mode screen with numbered callouts 1-17, showing waveform display, track information panel, action pads with BEAT SELECT, SELECT, and EDIT panels, and PALETTE panel on the right side]

2. Select [EDIT] mode.

3. Create a new project.
   The current project will be closed, and the screen will be cleared.

4. Save a project (page 14).

5. Render audio files (page 15).

6. Move the playback position to the beginning of the timeline.

7. Start or pause playback.

8. Turn on/off loop playback.
   When you start playback in Loop:ON (in orage), the selected range plays in loop.

9. Quantize
   Specify the minimum unit (1 beat/4 beats) to select the range.

10. Track information panel (page 7)

11. Bar number
    You can change the position of the enlarged waveform to be displayed by dragging to the right/left.

12. Enlarged waveform
    Click to specify the range.

13. Change a zoom ratio of the enlarged waveform.
    [Screenshot: arrow icon] : Hide the icons.
    [RST]: Reset the zoom ratio.

14. Display the browser panel (page 8).

15. Action pad [BEAT SELECT] panel (page 9)/[SELECT] panel (page 10)

16. Action pad [EDIT] panel (page 11)

17. Action pad [PALETTE] panel (page 12)

## Track information panel

- Following tracks cannot be loaded to the deck of EDIT mode.
  - Unimported tracks
  - Unanalyzed tracks
  - Tracks in the [Devices] category
- Active Censor marks, phrases, and vocal information cannot be displayed on the deck of EDIT mode.

[Screenshot: Track information panel with numbered callouts 1-7, showing artwork, track name "One Night Stand", selected range indicator, time/measure display, track time 03:55.2, key Gm, and BPM 117.57]

1. The track's artwork image is displayed.
   Point on the artwork image with the mouse cursor to display [Screenshot: eject icon], and then click [Screenshot: eject icon] to unload the track.

2. Track name

3. Selected range

4. Time and measure at the pointed position with the mouse cursor

5. Track time

6. Key

7. BPM

**Note**

- When you load a track, BPM is constantly adjusted. For tracks analyzed with [Dynamic] or adjusted partially with grid spacing in the [GRID] panel, some part of track will have a different sound or BPM than its original.

# Editing

After loading a track in [Collection] to the deck, you can duplicate, copy cut, delete, mute, etc. a specified range of waveform within the track.
The edited contents are not applied to the track in [Collection]. To use the edited contents for your DJ play, render the edited track as an audio file ( [Screenshot: render icon] ), and then add the audio file to [Collection].
Also, you can edit again after saving the edited contents as a project.

## Selecting a track for editing

[Screenshot: Full EDIT mode screen with browser panel open, showing Collection with track list including tracks by Steve Richmond and others, with genres, keys, and BPM columns visible]

1. Click the [BROWSER] tab to open the browser panel.

2. Select a tack from [Collection].
   Double-click a track you want to edit.
   - Information about Grid, Hot Cue, Memory Cue, and Active Censor are also loaded, but these information cannot be edited.
   - Information about phrases and vocal are not loaded.
   - A track edited in EDIT mode is displayed with [Screenshot: edit indicator icon].

3. Click the [BROWSER] tab again to close the browser panel.

## Selecting the range for editing

To select the range, drag on the enlarged waveform. Or, select the starting point, and then click the ending point with holding the Shift key. The bar length, beat length, and time of the selected range will be displayed.

[Screenshot: Enlarged waveform with bar numbers 27-30, showing selected range of 1.0 Bars, 4 Beats, 00:02]

### Specifying the range using the [BEAT SELECT] panel

You can specify the range with beats of number from the selected position.

[Screenshot: BEAT SELECT panel showing 8 buttons arranged in 2 rows: 1 BEAT, 2 BEAT, 4 BEAT, 8 BEAT (top row), 16 BEAT, 32 BEAT, 64 BEAT, 128 BEAT (bottom row)]

### Changing the length of the selected range using the [SELECT] panel

You can double or halve the length of the selected range.

[Screenshot: SELECT panel showing 3 buttons with numbered callouts 1-3: 1/2 HALF, x2 DOUBLE, and (X) CANCEL]

1. Halve the length of the selected range.

2. Double the length of the selected range.

3. Cancel the selected range.

### Editing a track using the [EDIT] panel

You can edit using the selected range.

[Screenshot: EDIT panel showing 8 buttons with numbered callouts 1-8: CLONE, COPY, PASTE, INSERT (top row), DELETE, CLEAR, UNDO, REDO (bottom row)]

1. Copy the selected range, and then insert after it.

2. Copy the selected range.

3. Paste the copied selected range.

4. Insert the copied selected range.

5. Delete the selected range from the track.

6. Mute the selected range.

7. Return to the previous state.

8. Return to the state before [UNDO].

**Hint**

- You can edit by selecting an item from the [Edit] menu in the upper-left corner of the screen or from the menu of right-clicking.

### Keeping the selected range in the [PALETTE] panel

Drag and drop the selected range to the [PALETTE] panel to keep it (up to 100 ranges for a track).

[Screenshot: PALETTE panel showing multiple kept waveform ranges with track name "One Night Stand", with numbered callouts 1-5 pointing to: hide panel arrow, delete button, track name label, waveform entries with play buttons, and add (+) button at bottom]

1. Hide the [PALLETE] panel. Click again to show the panel.

2. Change the screen to delete the kept range.
   Click the delete button of each range.

3. Name the title for the kept range.

4. Play/pause the kept range.
   You can also play the kept range by clicking the waveform.

5. Keep the selected range in the [PALETTE] panel.

### Inserting the selected range from the [PALETTE] panel

Drag the waveform in the [PALETTE] panel, and drop it to the point where you want to insert in the enlarged waveform.

**Hint**

- You can also select [Insert] from the [EDIT] menu in the upper-left corner of the screen.
- You can also select [Insert] by right-clicking the point where you want to insert in the enlarged waveform.

## Saving the edited contents as a project

If you save the edited contents as a project, you can select the project to edit from the continuation.

1. Click [Screenshot: save icon] in the upper-left corner of the screen.
   For the first time to save, the folder where the project is saved opens. Enter the project name, and then the edited contents are saved as a project.
   If these contents have already saved as a project, it is overwritten.

**Hint**

- You can also select [Save] from the [File] menu in the upper-left corner of the screen. To save with a different name from the already saved project, select [Save as].
- Project file is not supported with followings:
  - Cloud Library Sync
  - Library Backup
- When the track is loaded, a message to save the editing appears when you load the other track.

### Open the saved edited contents (project)

Click the project in the upper-left corner of the screen to display the recent project list. Projects are displayed in the new order not according to the folders including projects.
When you select the project, the track is automatically loaded to the deck, and then you can edit.

**Hint**

- When you select [Open a recently edited project] from the [File] menu in the upper-left corner of the screen, projects used recently are displayed in the list. Select [Open project] to open the folder where you saved the project at the last.
- The list for the recently used projects are displayed with up to 20 projects.

**Note**

- You cannot open the project of the track deleted from [Collection].

## Render the edited track as an audio file

For DJ play the track included with edited contents, render the track as an audio file to add to [Collection], Hot Cue, Memory Cue, grid information, Active Censor, etc. can be exported from the original track to render. However, analysis information of phrases and vocal cannot be exported. Please analyze again.

**Note**

- If you have signed up the Creative plan, rendering can be used.

1. Click [Screenshot: render icon] in the upper-left corner of the screen.
   The [Render Audio] window appears.

2. Enter the track name and artist name, and select the location to be saved, file format, and sample rate/bitrate.

[Screenshot: Render Audio dialog window showing fields for Track (New Project), Artist (Steve Richmond), Location of Files (path to Music/PioneerDJ/Edited Tracks with Browse button), File Format (.WAV dropdown), Sample Rate (44.1 kHz dropdown), and a checkbox "Show track in collection after render", with Render and Cancel buttons]

3. Click [Render].
   Rendering an audio file starts.
   Rendered audio file will be automatically added to [Collection].

**Hint**

- You can also select [Render audio files] from the [File] menu in the upper-left corner of the screen.

**Note**

- The track with 180 minutes or more cannot be rendered.
