# Introduction

Pad Editor can change Pad modes assigned to pad mode buttons on a DJ equipment supporting Pad Editor and customize the function for each pad.

**DJ equipment supporting Pad Editor**

- DDJ-FLX10
- DJM-S7
- DJM-S11
- DDJ-XP2
- DDJ-800
- DDJ-1000
- DDJ-XP1

Pad Editor can be used in PERFORMANCE mode.

# Using Pad Editor

Check that [PERFORMANCE] is selected in the upper-left corner of the screen.

## Connecting the computer to the controller supporting Pad Editor

Before using Pad Editor, connect the computer to the controller supporting Pad Editor with a USB cable, and then turn on the controller.
For details on the controller supporting Pad Editor, see the Instruction Manual of the controller.

**Connection example**

[Screenshot: Connection diagram showing a computer at the top connected via two USB cables to two DJ controllers below. The left controller is labeled "DJ controller unsupporting Pad Editor" and the right controller is labeled "DJ controller supporting Pad Editor (e.g.DDJ-XP2)"]

## Displaying the Pad Editor screen

When a controller supporting Pad Editor is connected, click [PAD] in the upper-right corner of the screen to display the Pad Editor screen.

[Screenshot: PAD EDITOR window showing ASSIGN 1 and ASSIGN 2 tabs at the top, PERFORMANCE MODE section with 8 pad mode dropdowns (HOT CUE, PAD FX1, BEAT JUMP, SAMPLER, KEYBOARD, PAD FX2, BEAT LOOP, USER 1), PERFORMANCE PADS section with TEMPLATES dropdown, and a 4x4 grid of 16 pads all set to HOT CUE labeled A through P. Numbers 1-4 indicate different sections of the interface.]

1. ASSIGN 1 / ASSIGN 2
   For each ASSIGN, 8 Pad modes can be set.

2. Pad modes
   When you change the Pad mode by using [v], the function is changed to be assigned to each pad.
   For details on each function, see "Pad modes list" (page 10).

3. Templates for customizing pads
   When you select from [USER1] to [USER8] for the Pad mode, select a template for customizing. (page 8)
   For details on each template for pad customizing, see "List of templates for pad customizing" (page 11).

4. Pad functions
   Displays Pad functions to be assigned according to the Pad mode.
   For details on customizing Pad functions, see "Customizing Pads" (page 8).
   For details on each assigned Pad function, see "List of assigned Pad functions for customizing" (page 12).

**Note**

- Pad Editor is applied to all decks.

## Customizing Pads

For selectable items, see "List of assigned Pad functions for customizing" (page 12).

1. Click [v] of the Pad mode in [PERFORMANCE MODE], and select from [USER1] to [USER8].

[Screenshot: PAD EDITOR window showing the first pad mode dropdown expanded with a list of options: HOT CUE, PAD FX1, BEAT JUMP, SAMPLER, KEYBOARD, KEY SHIFT, BEAT LOOP, PAD FX2, SLICER, ACTIVE CENSOR, SEQUENCE CALL, SP.SCRATCH, STEMS, USER 1 through USER 8]

2. Click [v] of [TEMPLATES] in [PERFORMANCE PADS], and select a template. A template to base Pad functions on is assigned to each pad.

[Screenshot: PAD EDITOR window showing the TEMPLATES dropdown expanded with a list of options: HOT CUE, PAD FX1, BEAT JUMP, SAMPLER, KEY SHIFT, BEAT LOOP, PAD FX2, ACTIVE CENSOR, SEQUENCE CALL, LIGHTING, TRANSPORT, SP.SCRATCH, STEMS, USER 1 through USER 8, Combination 1 through Combination 9]

3. Click [v] of the pad in [PERFORMANCE PAD], and select your preferred function.

[Screenshot: PAD EDITOR window showing the first pad's dropdown expanded with a list of function categories: HOT CUE, PAD FX1, BEAT JUMP, SAMPLER, KEY SHIFT, BEAT LOOP, PAD FX2, ACTIVE CENSOR, SEQUENCE CALL, LIGHTING, TRANSPORT, SP.SCRATCH, STEMS. Each category has a submenu arrow indicator.]

Select a detailed item depending on the function.

[Screenshot: PAD EDITOR window showing the first pad's dropdown with BEAT LOOP selected and its submenu expanded showing beat values: 1/64, 1/32, 1/16, 1/8, 1/4, 1/2, 1, 2, 4, 8, 16, 32, 64, 128, 256, 512]

# Menu list

## Pad modes list

| Pad mode | Description |
|---|---|
| HOT CUE | When using a Hot Cue, you can instantly call up the specific playback position. |
| PAD FX1 | While the pad is being pressed, the assigned FX is turned on. |
| BEAT JUMP | You can instantly jump to the playback position with the beat and direction assigned to the pad. |
| SAMPLER | Plays tracks loaded to the sampler slots. |
| KEYBOARD | Plays a Hot Cue with pitch assigned to the pad. |
| KEY SHIFT | Shifts the key among the track with the pitch assigned to the pad. |
| BEAT LOOP | Plays in loop set with the beat assigned to the pad. |
| PAD FX2 | While the pad is being pressed, the assigned FX is turned on. |
| SLICER | While the pad is being pressed, the track is sliced and plays the sliced section in loop. |
| ACTIVE CENSOR | By using the ACTIVE CENSOR, you can apply the effect or function to the specified section. When the playback position reaches the specified position, the effect or function is automatically turned on. |
| SEQUENCE CALL | Plays the saved sequence. |
| SP.SCRATCH | Use this feature to take the track loaded into the sampler slot and load it onto a deck. |
| STEMS | Use this feature to divide a track into VOCAL, INST, BASS, and DRUMS Stems and output the audio accordingly. |
| USER 1 to 8 | Use this mode to customize Pad functions. |

## List of templates for pad customizing

| Template name | Description |
|---|---|
| HOT CUE | Assigns the Hot Cue A to P. |
| PAD FX1 | Assigns the Pad FX. |
| BEAT JUMP | Assigns the beat and direction of the jump. |
| SAMPLER | Assigns Sampler Slot 1 to 16. |
| KEY SHIFT | Assigns the pitch for the Key Shift. |
| BEAT LOOP | Assigns the beat for the Beat Loop. |
| PAD FX2 | Assigns the Pad FX. |
| ACTIVE CENSOR | Assigns effects and functions for the ACTIVE CENSOR. |
| SEQUENCE CALL | Assigns a slot for the sequence playback. |
| LIGHTING | Assigns items which can be controlled on LIGHTING. |
| TRANSPORT | Assigns the functions (changing the playback position, playback/pausing, Hot Cue settings, etc.). |
| SP.SCRATCH | Assigns Sampler Slot 1 to 16. |
| STEMS | Assigns Active Stem(Vocal, Inst, Bass, Drums) and Active Stem FX(Vocal, Inst, Bass, Drums). |
| USER 1 to 8 | Assigns the customized Pad functions. |
| Combination 1 to 9 | Assigns a recommended combination for Pad functions. |
