### Operation panel

#### [1 PLAYER] layout

[Screenshot: Operation panel for 1 PLAYER layout showing vertically arranged controls labeled 1-5: track skip buttons (1), jump forward/backward buttons (2), beat count selector (3), CUE button (4), and play/pause button (5)]

#### [2 PLAYER] layout

[Screenshot: Operation panel for 2 PLAYER layout showing vertically arranged controls for two players, each with CUE button (4), play/pause button (5), and a crossfader (6) between them]

#### [DUAL PLAYER] layout

[Screenshot: Operation panel for DUAL PLAYER layout showing controls for two players with jump buttons (2), beat count selector (3), CUE button (4), play/pause button (5), crossfader (6), A-GAIN controls (7), 3 BAND EQ HIGH/MID/LOW (8), and DUAL CONTROL button (9)]

1. Move to the beginning of the track (track search). ([1 PLAYER] layout)
   - >>|: Move to the next track.
   - |<<: Move to the beginning of the track currently playing. Click twice to move to the previous track.

2. Jump to a different playback position. ([1 PLAYER] layout, [DUAL PLAYER] layout)
   - >: Jump forward from the current playback position.
   - <: Jump backward from the current playback position.

3. Set the beat count or bar to jump from the current playback position when < or > is clicked. ([1 PLAYER] layout, [DUAL PLAYER] layout)

4. Set the Cue position. (page 94)

5. Start or pause playback.

6. Adjust the audio mix balance of outputs from Players A and B (crossfader). ([2 PLAYER] layout, [DUAL PLAYER] layout)

7. Adjust the volume of each player ([AUTO GAIN]). ([DUAL PLAYER] layout)

8. Mutes each player's band ([3 BAND EQ]). ([DUAL PLAYER] layout)

9. Turn on/off [DUAL CONTROL] that plays/pauses on the Player A and B at the same time. ([DUAL PLAYER] layout)

### Playback track sub-information panel

Information of tracks loaded in the player panel, comments for [MEMORY] and [HOT CUE], etc. are displayed.
Click an item to display its panel on the playback track sub-information panel.

[Screenshot: Playback track sub-information panel showing a list of cue/loop points with timestamps (00:02:286, 00:11:205, 00:38:406, 00:45:095, 00:57:134, 01:03:823, 01:06:945, 01:17:647, 01:31:400, 01:34:592) with colored dots and loop icons, and three tabs at the bottom: MEMORY (1), HOT CUE (2), INFO (3)]

1. [MEMORY] (page 70)

2. [HOT CUE] (page 71)

3. [INFO] (page 72)

#### [MEMORY] panel

Saved cue and loop points are displayed. To save a cue, see "To save a cue point" (page 95). To save a loop point, see "To save a loop point" (page 97).

[Screenshot: MEMORY panel showing a list of cue/loop points with timestamps and colored dots, with numbered labels 1-5 indicating: color indicator (1), time display (2), comment field (3), loop indicator (4), and delete button (5). Tabs at bottom show MEMORY selected, HOT CUE, and INFO]

1. Change the color of the cue marker shown on the upper side of the waveform display.
   Depending on the DJ player, the changed color can be displayed.

2. Shows the time of a cue or loop point (minute/second/millisecond).
   Click it to move the playback point to the cue or loop point.

3. Enter a comment for a cue or loop point to display.
   Depending on the DJ player, the edited comment can be displayed.

4. Shows when a loop point is saved. Click it to set to an active loop. Its color then changes to red.

5. Delete a cue or loop point.

#### [HOT CUE] panel

Hot Cues are displayed. For information about Hot Cues, see "Using a Hot Cue" (page 98).

[Screenshot: HOT CUE panel showing Hot Cues labeled A through H with timestamps (A 00:07, B 00:10, C 00:36, D 01:20, E 01:05, F 01:42, G 02:11, H 00:31) with colored letter indicators (green for cues, orange for loops), numbered labels 1-5. Tabs at bottom show MEMORY, HOT CUE selected, and INFO]

1. Hot Cues are saved. If its color is green, a Cue is saved. If its color is orange, a loop is saved.
   You can change the color of the Cue marker by right-clicking it.

2. Shows time of a Hot Cue point (minute/second).
   Click it to jump to the saved Hot Cue point.
   If it is empty, click it to save the Hot Cue point.

3. Enter a comment for a Hot Cue to display.

4. Shows when a loop point is saved.

5. Delete a Hot Cue.

### Function panel ([1 PLAYER] layout)

Switch the display to [CUE/LOOP] or [GRID].

[Screenshot: Function panel selector showing two options: CUE/LOOP (1) and GRID (2) with Hot Cue buttons A, B, C visible]

1. [CUE/LOOP] (page 74)

2. [GRID] (page 75)

#### [CUE/LOOP] panel

Set and save a Hot Cue, loop, and cue.

[Screenshot: CUE/LOOP panel showing a horizontal bar with numbered labels 1-7: Hot Cue buttons A through H with colored indicators (1), MEMORY button (2), cue/loop navigation arrows with delete button (3, 4), beat loop controls with < and > arrows (5), quantize button (6), and menu button (7)]

1. Save Hot Cues. To delete a Hot Cue, point on it with the mouse cursor, and then click [X]. (page 99)

2. Save the cue and loop point which you set. (page 95) (page 97)

3. Move to a cue or loop point.
   - >: Move to the next cue or loop point.
   - <: Move to the previous cue or loop point.

4. Delete a cue or loop position.

5. Click the lower bar to switch the buttons.
   - < 4 >: Specify a beat length from the current position, and start the loop play. (Auto Beat Loop) (page 97)
   - IN OUT RELOOP: Set real-time cues and loop in/out points, and cancel/resume a loop playback. (Manual Loop) (page 94)

6. Turn on/off the quantize. (page 92)

7. Display a menu screen to set the color of the enlarged waveform, analyze the currently playing track, export tracks, etc.

#### [GRID] panel

Adjust the beat grid and phrase. For more details on the beat grid, see "Adjust the beat grid" (page 90).

[Screenshot: GRID panel showing a horizontal bar with numbered labels 1-17: beat position marker (1), BPM display showing "114.50" (2), TAP button (3), beat grid adjustment buttons for fine-tuning (4-9), x2 and x1/2 buttons (10-11), undo/redo buttons (12-13), metronome controls (14-15), lock button (16), and PHRASE EDIT section with CUT button, phrase type selector showing "VERSE5", and CLEAR button (17)]

1. Shift the playback position (a red vertical line at the center of the waveform) to the first beat of bar.

2. Display the distance of beat grid at the playback position (a red vertical line at the center of the waveform) as BPM. You can also enter BPM manually to change the distance of beat grid.

3. Change the beat grid according to BPM determined by the click interval of [TAP].

4. Move the beat grid to the left by 1 msec.

5. Move the beat grid to the right by 1 msec.

6. Widen the beat grid distance by 1 msec when [grid adjust] is turned on.

7. Narrow the beat grid distance by 1 msec when [grid adjust] is turned on.

8. Double the BPM. The length of each beat grid distance is halved.

9. Halve the BPM. The length of each beat grid distance is doubled.

10. Select the whole track as an adjustment range of the beat grid.
    When switching to the [GRID] panel, the whole track is already selected and this button is not available. The adjustment range is changed to the whole track when [grid adjust] is turned on and the range is set to the current position or later.

11. Re-set the beat grid from the current position (a red vertical line at the center of the waveform) according to the BPM determined.

12. Undo your last action in the [GRID] panel.

13. Redo the action you undid in the [GRID] panel.

14. Turn the metronome sound on/off.
    Metronome sound starts with the playback track, synchronized with the beat grid.

15. Change the metronome volume to one of three levels.

16. Enable/Disable Track Analysis, Key Analysis, and Beatgrid Adjustment. (page 92)
    If you set it to disable, a lock icon is displayed on the track list.

17. Edit each phrase which is displayed in the track information panel by loading the track analyzed with [Phrase].
    - After you click [CUT] to light up, click on the phrase to be divided.
    - To change the phrase type, click the phrase type selector.
    - To delete all phrases, click [CLEAR].
    - To adjust the position of the phrase, click the phrase and drag the divided position (white line).
    - To display the [PHRASE EDIT] panel, open the [Preferences] window > [View] tab > [Layout], and select [Phrase (Full Waveform)] (page 223). For details about [PHRASE EDIT] panel, refer to "PHRASE EDIT operation guide" on the rekordbox website.

[Screenshot: Function panel for 2 PLAYER layout showing a horizontal bar with numbered labels 1-8: beat grid adjust buttons (1), MEMORY button (2), beat loop/manual loop controls (3), MT (Master Tempo) button (4), RST (Reset) button (5), tempo slider with - and + buttons showing "118.72" (6), pitch bend up/down arrows (7), and quantize button (8)]

## Browser panel

For details about operations on the [Collection] screen, see "[Collection]" (page 7).

[Screenshot: Browser panel showing the full interface with numbered labels 1-19: shortcut area (1), tree view/media browser showing Collection, All, Date Added, Genre, Artist, Album (2/13/14/15), playlist palettes/tag list area (2), column headers for sorting (3), playlist path (4), PLAYER A selector (5), artwork column (6), track list with columns for Track Title, Artist, Album, Genre, BPM, Rating (7/8), key compatibility highlighting (5), artwork display toggle (6), category filter (7), track filter (8), search filter (9), playlist palette show/hide (10), My Tag button (11), RELATED TRACKS button (12), SYNC MANAGER button (13), Mobile Library Sync button (14), media browser tree (15), track list area (16), and additional controls (17/18/19)]

1. Display the shortcut screen. Drag a folder or playlist on the tree view to create a shortcut.

2. Display tag lists or playlists.
   3 of the 4 palettes on the right are playlist palettes. The left-most palette is used as Tag List.

3. Sort track information to search for a track. You can display/hide columns and change their order.

4. Display the playlist path, and select one to display the playlist.

5. Highlight the keys of tracks that are compatible with the key of the track on the selected deck when displaying the [Key] column. (Traffic Light)

   [Screenshot: Key column showing Traffic Light feature with keys "Ebm" (white), "A" (green highlighted), and "Cm" (green highlighted)]

6. Change the display pattern for the artwork.
   - Upper artwork icon: Display the upper part of the artwork.
   - Full artwork icon: Display the whole artwork.

7. Search for a track, using a category filter.

8. Search for a track by refining with [BPM], [KEY], [RATING], [COLOR], and [MY TAG]. (Track filter)

9. Search for a track, using the search filter.

10. Show/Hide playlist palettes.

11. Display the My Tag settings screen. (My Tag)

12. Display the RELATED TRACKS screen.

13. Display the [SYNC MANAGER] window. Set tracks in rekordbox to be synchronized with external libraries. (page 49)

14. Display the [Mobile Library Sync] window. Set pairing with rekordbox (iOS). (page 58)

15. Select to display on the track list. ([Media Browser])
    - [Collection]: Display all tracks in rekordbox.
    - [Playlists]: You can organize the collection on rekordbox by creating a playlist prior to your DJ performance.
    - [Related Tracks]: Display RELATED TRACKS of the loaded track.
    - [Track Suggestion]: Display the list of recommended tracks for the track you have loaded.
    - [Hot Cue Bank Lists]: You can save 8 Hot Cue points with different tracks. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide. (page 100)
    - [iTunes]: Refer to the iTunes library. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [Apple Music]: Refer to tracks on Apple Music. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [Inflyte]: Refer to tracks on Inflyte. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [Beatport]: Refer to tracks on Beatport. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [TIDAL]: Refer to tracks on TIDAL. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [rekordbox xml]: Refer to xml with the rekordbox xml format. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [Explorer]: Display folder layers of your computer and external devices such as a USB flash drive. Click the [Preferences] window > [View] category > [Layout] tab and set to display/hide.
    - [Devices]: Display USB storage devices, SD cards, and mobile devices.
    - [Histories]: Display playback histories on DJ equipment.
    - [Recordings]: Display files recorded on rekordbox.

16. Display tracks in a playlist selected from the tree view. (Track List)

17. Display the sub browser window. (page 12)

18. Display the track information screen.

19. Display the [Track Suggestion] window.

### Link status panel

The [LINK] button is displayed when rekordbox is connected to the DJ equipment supporting PRO DJ LINK by a LAN cable or wireless LAN. Click the [LINK] button to start communication with the DJ equipment. Information on the DJ equipment on the same network is displayed in the Link status panel. For more details, see "Using the LAN connection for DJ performance (LINK EXPORT)" (page 85) and "Using the LAN connection" (page 109).

[Screenshot: Link status panel showing numbered labels 1-5: LINK button with MASTER indicator (1), BPM display showing "120.00" with adjustment controls (2), MASTER sync indicator (3), sync icon (4), and four deck status indicators showing tracks "In And Out Of My Life", "Eternal Dance", "One Of A Kind", and "One Night Stand" with MASTER SYNC labels and green status dots (5)]

1. Start/terminate communication with the DJ equipment connected by a LAN cable or wireless LAN.

2. Adjust the tempo (BPM) when rekordbox is set as the Sync Master. (Master BPM)

3. Synchronize using the master tempo (BPM) specified by rekordbox when using the beat sync function on a DJ player. (Sync Master)

4. Load the current master tempo (BPM).

5. Display the status of each DJ equipment connected by a LAN cable or wireless LAN. (Link Status)
