Build ordered tracklists for DJ sets using cached audio analysis and enrichment data. The agent scores transitions across 6 axes (key, BPM, energy, genre, brightness, rhythm), sequences tracks using beam search, and presents multiple candidates for you to compare and refine.

<Aside type="note">
  Set building is read-only — it never modifies track metadata. It only creates playlist orderings exported as XML.
</Aside>

## Prerequisites

- **Genre classification** should be substantially complete for the tracks you want to use.
- **Cache coverage at 100%** across all providers. Run `cache_coverage` to check.
- **Essentia** is optional but improves energy, brightness, and rhythm scoring.

## Transition scoring axes

Each transition between adjacent tracks is scored on 6 axes:

| Axis | What it measures |
|---|---|
| **Key** | Harmonic compatibility via Camelot wheel position |
| **BPM** | Tempo proximity (smaller jumps score higher) |
| **Energy** | Energy level continuity and adherence to the target curve |
| **Genre** | Genre affinity (same genre or compatible genres score higher) |
| **Brightness** | Spectral similarity (smooth tonal transitions) |
| **Rhythm** | Rhythmic pattern compatibility |

The **priority axis** parameter weights one axis more heavily. "Balanced" gives equal weight to all six.

## Energy curves

### warmup → build → peak → release (default)

| Phase | Position | Energy |
|---|---|---|
| Warmup | 0-15% | Low energy, atmospheric |
| Build | 15-45% | Gradually increasing |
| Peak | 45-75% | Maximum energy |
| Release | 75-100% | Winding down |

### flat

Constant peak energy throughout. Useful for peak-time sets.

### peak_only

Minimal warmup (0-10%), extended peak (10-85%), brief cooldown (85-100%).

### Custom

Specify energy levels per track position. Tell the agent the shape you want.

## Tips

- **Start with a larger pool than you need.** A 12-track set benefits from 40-60 candidates.
- **Pin your opener and closer first.** The agent optimizes the middle.
- **Use genre focus loosely.** "Techno, Tech House" gives better results than "Techno" alone.
- **Re-run with different priority axes.** A "harmonic" build and an "energy" build can look very different.
- **Check individual transitions.** Use `score_transition` on any two tracks to see the detailed axis breakdown.
